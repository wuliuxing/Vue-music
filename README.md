

# 运行
```
cnpm / yarn install

cnpm / yarn run serve
```

# 简介
项目通过Vuecli开发的一个webapp，数据暂时存储在Vuex中，播放器调用vue-aplayer，图标主要用的是ElementUI。
项目还在不断完善中，功能会继续开发
